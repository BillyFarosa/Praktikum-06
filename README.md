Penjelasan Program :

    Fungsi "T)ambah" : *Input huruf "t" untuk menjalankan fungsi "T)ambah". *Fungsi "T)ambah" berfungsi untuk menambahkan Data pada List data yang ada. *Data yang dimasukan akan otomatis masuk juga kedalam List.

    Fungsi "E)dit" : *Input huruf "e" untuk menjalankan fungsi "E)dit". *Lalu input NAMA data yang ada pada List untuk dirubah datanya. *Fungsi "E)dit" berfungsi untuk merubah Data yang ada pada List sebelumnya.
